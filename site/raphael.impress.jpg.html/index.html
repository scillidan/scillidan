    Directory Opus + CMDER¶

    移动例外文件，核对行数和文件数

    artist：创建文件夹,移动相应的图片到文件夹,写入artist元数据,删除文件名中的artist信息

    move ".artist*.*" ".dir"/

    exiftool -artist=".artist" ".path/."

    renamer --find ".artist_" ".path/*"

    查找png、tif等除jpg mkdir "\(1" && move "\)1*." "\(1" && cd "\)1" && exiftool -artist="\(1" "\)1." && renamer --find "$1_" * && trash "_original"
